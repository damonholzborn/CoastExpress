Coast Express

?

Welcome to Coast Express, a MIDI settings utility for the Make Noise 0-Coast.

System Requirements

Coast Express is currently in beta. These requirements will be updated as further testing is completed.

  • Chrome is currently the only officially recommended browser, though Opera and Edge should also work. All other browsers will not work.
  • Testing has only been done on macOS, but it should work in Windows and Linux as well. Some browsers on Android might work as well.
  • You will need a MIDI interface connected to your computer.

Intructions

  1. Connect your MIDI interface before loading the Coast Express page.
  2. Choose the MIDI port of your 0-Coast from the pull-down menu.
  3. Change settings using the buttons and sliders.

Note that the settings are not automatically saved on the 0-Coast so if you turn the power off the settings will be lost. To save settings on the 0-Coast, hold PGM A to enter the Program Pages, then hold PGM B to save.

Coast Express for Max

Coast Express is also available in a version for Cycling '74's Max. The Max version provides a few extra widgets not (yet?) available for the web version:
  • An onscreen keyboard to send MIDI notes.
  • A dial to send Mod Wheel.
  • MIDI clock out.
  • Interface is reflected in Mira for display on an iPad.
Download the Max patch.
